Html Paragraphs

HTML Paragraphs is a text block that always starts with a new line and to which browsers automatically add some margin before and after.
In HTML large/small screens or resized windows will create different results.
We can not add extra spaces/lines to change the look of our HTML code as the browser will remove extra spaces/lines when the page is displayed.
To define a break in a line without changing the paragraph, we use an empty element called <br> tag.
