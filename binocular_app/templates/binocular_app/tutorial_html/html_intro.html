Html Intro

HTML i.e. Hyper Text Markup Language is the standard markup language for web pages, using which we can create our own website.
It informs a web browser of the structure of the web pages though a sequence of elements .
A html element consist of a start tag, content and end tag.
HTML elements can be nested
Though not all tag have a end tag.

Start TagContentEnd tagPurpose
<p> text </p>text as paragraph
<i> text </i>text in italic
<a> linking item </a>create a link
<br> - -break the line
HTML tags are not case sensitive.
The browser only displays the element content.
