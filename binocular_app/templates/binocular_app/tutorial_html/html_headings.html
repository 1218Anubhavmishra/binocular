Html Headings

HTML Headings are titles that we want to show in our webpage.
They are defined using multiple tags ranging in a pattern of decreasing importance from <h1> being highest to <h6> being lowest.
Heading are often used to show structure for better understanding of the provided content.
But, Use HTML headings for headings only. Don't use headings to make text BIG or bold .

Each HTML heading has a default size. However, you can specify the size for any heading with the style attribute, using the CSS font-size property :
Example - < h1 style= "font-size: 60px; "> Heading 1 </ h1>
