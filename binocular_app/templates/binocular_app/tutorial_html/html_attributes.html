Html Attributes

HTML attributes are additional information about HTML elements and are specified in start tag in a name value pair.
Eg. name = " value " .

HTML standard does not requires lowercase attribute names but it is still recommended that way.
But demanded in lowercase for stricter document types like XHTML.

HTML standard does not requires quotes around attribute values but it is still recommended that way.
As un-quoted attribute values having spaces in them are often displayed uncorretly.
Eg, alt = " logo image "
And also it is demanded in quotes for stricter document types like XHTML.

Double quotes around attribute values are the most common in HTML, but single quotes can also be used.

In some situations, when the attribute value itself contains double quotes, it is necessary to use single quotes.
Eg. title = ' Carol "Avenger" Danvers '
