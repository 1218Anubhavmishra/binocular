Html Quotation

The blockquote element defines a section that is taken from else where and is indicated with indentation.
The q element defines a short quotation that is indicated with quotation marks aroung it.
The abbr element defines an abbrevaition, acronym or useful information about the element when we hover our mouse over the element and indicated using the global title attribute.
The address element defines the contact information of the owner in their pages and indicated in italic with a line break before and after the element.
The cite element defines the title of a creative work and indicated in italic.
The bdo element is used to override the current text direction.
