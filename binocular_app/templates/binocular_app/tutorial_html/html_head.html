Html Head

HTML head element is a container tag for various document details and is placed within html tag before the body tag.
The title element is a required in html as it is usec by search engines algos to in indicate the page.
The style element is used to define internal style information for that webpage.
The link element is used to connect current webpage to external resourses.
The script element is used to define client-side Javascript.
The base element specifies the base URL or target url for all relative URL page.

The meta element is used to display Metadata, which are important webpage details like character set.

These are used by browser but not displayed on the page.
The viewport is the user's visible area of a webpage and varies with the device.
It's initial-scale part sets the initial zoom level of the web page.
The width part can be given the value 'device-width' , that allows the page width to follow the screen-width .
