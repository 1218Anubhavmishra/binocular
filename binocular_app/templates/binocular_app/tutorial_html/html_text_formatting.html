Html Text Formattings

HTML formatting elements are designed to text in following special ways :
<b> for text with boldness, but no extra importance.
<strong> for text with strong importance.
<i> for text with importance or alternate behavior.
<em> for emphasised text.
<small> for smaller text.
<mark> for marked or highlighted text.
<del> for text that is deleated, usually by a striking line through it.
<ins> for text that needs to be inserted , indicated with a underline.
<sub> for text that is shown half a character below the usual text.
<sup> for text that is shown half a character above the usual text.
