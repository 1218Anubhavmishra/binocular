Html Links

HTML links are hyperlinks that on being clicked will take you to another pages.
When we hover over the link,the mouse arrow turns into a hand.
The HTML <a> tag defines a hyperlink which can be a text, an image or any other element.

The href attribute indicates the link's destination.
URLs can be of two types :
Absolute - complete address of the connected document.
Relative - address relative to the current page.
The link text is the part that will be visible to the reader.
An unvisited link is underlined and blue.
A visited link is underlined and purple.
An active link is underlined and red.

By default, the linked page will be displayed in the current browser window,but it can be changed with target attribute.
_self - Opens the document in the same window/tab as it was clicked.
_blank - Opens the document in a new window or tab.
_parent - Opens the document in the parent frame.
_top - Opens the document in the full body of the window.

To use an image as a link, just put the <img> tag inside the tag.
To specify link to send emails we use mailto: inside href attribute.
Eg, <a href="mailto: username@domain" > send mail </a>

HTML buttons can also be used as links by using some JS code.
The title attribute specifies extra information about an element, which is most often shown as a tooltip text when the mouse moves over the element.

HTML links can be created bookmarks using id attribute to jump to specific parts in a web page.
link to that bookmark is created by specifying the '#' and id_value as url.
eg, <p id="bookmark" > reach here </p>
<a href="#bookmark" > go to bookmark </a>
