Python Intro

Python is a popular programming language which can be used for-
  • server to create web applications.
  • Enables rapid prototyping for Production ready software development.
  • Connect with Database to read and modify files.
  • Handle big data to perform complex mathematical calculations.

Why choose Python ?

  • Python works on different platforms.
  • Has a simple syntax just like a english language.
  • Its syntax allows creation of programs with fewer lines.
  • It runs on an Interpreter which enables execution just after writting it.
  • Python can used to program Procedural way or an Object Oriented way or even a functional way.
  • Python allows new line to complete a command unlike other programming languages which often use semicolan or parenthesis for the purpose.
  • And also allow indentation using white spaces to define a scope of loops ,statements, functions and classes unlike other programming languages whick mostly use curly-braces for this purpose.
